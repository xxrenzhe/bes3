推荐阅读顺序：

1. 先看架构与实施总览，确认模块边界和一键工作流。
2. 再看环境变量与第三方依赖，完成本地、测试、生产配置。
3. 最后看部署与运维，按容器或主机模式上线，并处理媒体持久化和备份。
4. 上线后参考 CI_CD与监控 了解自动构建流程、回滚方案和监控巡检。
5. 如果需要判断《需求分析》中的方法论该如何映射到 Bes3，最后看产品定位映射文档。
